To run the Jupiter notebook:
python.org installation + windows:
	1) open the command prompt - windows key (search), type in cmd, press enter
	2) check to make sure python is properly installed by typing "python --verison" without the quotes in the command prompt
		you could get a version back ex: python 3.7.5
	3) type "pip install jupyter" without quotes in the command prompt
	4) type "jupyter notebook" without the quotes in the command prompt
	5) navigate to where you downloaded the file in the opened window

Anaconda installation + windows:
	1) open the command prompt - windows key (search), type in cmd, press enter
	2) check to make sure python is properly installed by typing "python --verison" without the quotes in the command prompt
		you could get a version back ex: python 3.7.5 (anaconda)
	3) type "jupyter notebook" without the quotes in the command prompt
	4) navigate to where you downloaded the file in the opened window

python.org installation + mac/linux:
	1) open the command prompt - cmd/windows key + space (search), type in terminal, press enter
	2) check to make sure python is properly installed by typing "python --verison" without the quotes in the command prompt
		you could get a version back ex: python 3.7.5
	3) type "pip install jupyter" without quotes in the command prompt
	4) type "jupyter notebook" without the quotes in the command prompt
	5) navigate to where you downloaded the file in the opened window

to run python using a text editor:
	save your file as file.py
	navigate to your file in the command prompt
		windows:
			cd folder -> goes into the folder
			cd .. -> goes up a folder
			dir -> list everything in current folder
			python file.py -> runs file.py
		Mac/Linux:
			cd folder -> goes into the folder
			cd .. -> goes up a folder
			ls -> list everything in current folder
			python file.py -> runs file.py
